## what is a NUI?
A NUI (Native User Interface) in the context of FiveM is like having your own customizable website integrated into your GTA 5 view. It empowers you to freely edit and design it as if you're working on your personal web page using HTML, CSS, and JS.

HTML, CSS, and JS are the three essential languages used for creating websites. HTML serves as the foundation, defining the structure of a webpage. CSS adds style and visual aesthetics to the website, while JS handles the technical aspects, making the website interactive and functional.

Before delving into NUI creation, it's essential to grasp the basics of CSS, JS, and HTML. In this wiki, we will provide a comprehensive overview of these coding languages, equipping you with the necessary knowledge to excel in crafting your own NUI.
